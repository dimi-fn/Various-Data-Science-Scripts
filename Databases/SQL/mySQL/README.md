# General

* useful commands for the mySQL server terminal

|**Command**|**Description**|
|--------|---------|
| `show databases` | displays the existed databases |
| `use database <db-name>`| use the desired database to query and edit|
|  `drop database <db-name>`  |    delete a database|
|    |    |
